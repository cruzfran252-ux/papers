# 📚 Papers Repository

Repositorio centralizado para organizar papers y artículos académicos con sus documentos y referencias relacionadas.

## 📁 Estructura

```
papers/
├── paper-title-1/
│   ├── README.md          (información del paper)
│   ├── main-paper.pdf     (opcional)
│   └── related-docs/      (carpeta para documentos relacionados)
├── paper-title-2/
│   └── ...
```

## 📝 Cómo agregar un nuevo paper

1. Crea una carpeta con el nombre del paper: `papers/nombre-del-paper/`
2. Dentro crea un `README.md` con:
   - Título
   - Autores
   - Año de publicación
   - Links a documentos
   - Resumen/notas

### Template README.md para cada paper:

```markdown
# [Título del Paper]

**Autores:** Nombre 1, Nombre 2  
**Año:** 2024  
**DOI/Link:** [Link al paper]

## 📄 Documentos

- [Paper Principal](link-al-pdf)
- [Documento Relacionado](link)
- [Código/Recursos](link)

## 📋 Resumen

Resumen breve del paper...

## 🔗 Referencias

- Referencia 1
- Referencia 2

## 📌 Notas Personales

Tus anotaciones aquí...
```
